# Enterprise Admin UI

A modern SaaS-style admin dashboard built with React, TypeScript, and Supabase.

This project is a production-grade application, focusing on real-world architecture, security, and UX patterns.

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- React Router
- TanStack Query
- Supabase (Auth, Database, RLS)

## Features (Current)

- Authentication (login/logout)
- Protected routes
- Secure Tickets module with Row Level Security
- View tickets
- Create tickets
- Inline update (status, priority)
- Delete tickets
- React Query caching and mutations

## In Progress / Planned

- UX polish (optimistic updates, disabled states)
- Search and filtering
- Pagination
- Multi-tenant organizations
- Role-based access

## Local Development

```bash
git clone https://github.com/alfredulu/enterprise-admin-ui.git
cd enterprise-admin-ui
npm install
npm run dev

```
